Energy-Aware Secure Data Transmission in Wireless Sensor Networks Using VAWGAN and NPOA

Project Objective

This project aims to improve secure communication in Wireless Sensor Networks (WSNs). VAWGAN is used to identify malicious sensor nodes, while NPOA selects the most energy-efficient and secure routing path. The proposed approach helps reduce energy consumption, improve packet delivery, and increase network lifetime.

Project Modules

  • Login Authentication
  • Create Wireless Sensor Network
  • Generate Sensor Nodes
  • VAWGAN Malicious Node Detection
  • NPOA Optimal Route Selection
  • Secure Data Transmission
  • Performance Analysis
