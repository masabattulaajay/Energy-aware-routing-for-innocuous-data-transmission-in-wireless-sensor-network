This project demonstrates secure and energy-efficient data transmission in Wireless Sensor Networks (WSN) using VAWGAN for malicious node detection and NPOA for optimal routing.

Project Features

  • Create Wireless Sensor Network
  • Generate Sensor Nodes
  • Detect Malicious Nodes using VAWGAN
  • Select Best Route using NPOA
  • Secure Data Transmission
  • Performance Analysis
